# VSO-MAS

> A collaborative Multi-Agent System framework for Visual Semantic Operations.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Overview

**VSO-MAS** (Visual Semantic Operations - Multi-Agent System) is a collaborative multi-agent framework designed for visual semantic understanding and intelligent task execution. This repository hosts the official product website, built with modern web technologies to showcase the framework's capabilities, documentation, and interactive demos.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) — React-based full-stack framework for production-grade applications
- **Build Tool**: [Vite](https://vitejs.dev/) — Next-generation frontend tooling for fast development and optimized builds
- **Styling**: Modern CSS / Tailwind CSS (configurable)
- **Deployment**: Static site generation (SSG) / Server-side rendering (SSR) ready

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm / npm / yarn

### Installation

```bash
# Clone the repository
git clone git@github.com:howardhenrystephen/vso-mas.git
cd vso-mas

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

### Build for Production

```bash
pnpm build
```

### Static Export

```bash
pnpm export
```
